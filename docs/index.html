Supplementary code and analysis for publication.

Publication figures

  • Figure 1. High levels of H3K27m3 and H2AUb broadly cover the naïve pluripotent genome.
  • Figure 2. X chromosome in naïve hESCs accumulate exceptional amounts of H3K27m3 without a globally repressive function.
  • Figure 3. H3K27m3 is adaptive to gene expression changes between naïve and primed pluripotent state and contributes to repression of non-state specific genes.
  • Figure 4. Loss of H3K27m3 in naïve hESCs derepresses a set of genes that drive hESCs towards a rare naturally ocurring intermediate pluripotent state.
  • Figure 5. Loss of H3K27m3 in naïve hESCs activates trophectoderm and placental gene expression programs.

Main data files

A large proportion of the figures shown in this publication can be drawn directly from values in the main tables:

  • Gene table. Contains histone mark values across 5kb regions around gene TSS in the hg38 genome, RNA-seq plus statistics on differentially decorated promoters and differentially expressed genes across conditions. differentially decorated regions.
  • Bins table. Gathers histone mark values across 10kb windows in the hg38 genome, plus statistics on differentially decorated regions across conditions.
